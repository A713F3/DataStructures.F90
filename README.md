# Linked List Module for Fortran

## Compile:
```
% gfortran -c linked_list.f90
% gfortran -c linked_list_example.f90
% gfortran -o main.out linked_list.o linked_list_example.o
% ./main.out
```

## Example Output:
```
List1:
node 1 :  1
node 2 :  2
node 3 :  3

List2:
node 1 :  1
node 2 :  6
node 3 :  9
node 4 :  7
node 5 :  5
```
